## 🌟 **Project Overview**

**Water Quality Monitoring (WQM)** is an enterprise-grade full-stack web application designed primarily as a Civic Complaint Management System. By tightly coupling a highly interactive React.js frontend with an absolute, robust Java Spring Boot architectural backend, the system guarantees efficient reporting and resolution of critical water hygiene issues.

### 🔥 **Hot New Features**

- 🚀 **One-Click Deploy** - The entire infrastructure can execute via automated `.bat` scripts.
- 🗄️ **Zero-Config Database** - Migrated natively to **Embedded H2** (Zero MySQL installation required!).
- 📊 **Administrative Core** - Interactive real-time metrics generated using `chart.js`.
- 🎓 **Premium Pedigree** - Fully bundled with high-quality A4 Final Year Documentation protocols.

### 🎯 **Key Features**
 
- 📝 **Civilian Grievances** - Instantly report systemic water contamination pipelines.
- 📱 **Fully Responsive** - A seamless interface adapting to all mobile & desktop hardware.
- 🎭 **Educational GUI** - Immediate onboarding regarding United Nations SDG 6 directives.
- ⚡ **RESTful Speed** - Lightning-fast decoupled interactions via API Axios fetching.
- 🌐 **System Thinking Models** - Designed around Causal Loop semantics prioritizing resolution cycles.

## 📦 **Installation & Setup**

### **Prerequisites**
- **Java Platform (JDK)** 11 or higher.
- **Node.js** and **NPM**.

### **Magic Launch (Fastest Way)**
Navigate to the root directory and simply double-click the execution scripts:
1. Double-click `run-backend.bat`. Wait 10 seconds.
2. Double-click `run-frontend.bat`. The browser will launch `localhost:3000`.

### **Manual Terminal Setup**

```bash
# 1. Launch the Backend (Spring Boot)
cd backend
mvnw spring-boot:run
# Listening strictly on http://localhost:8081

# 2. Launch the Frontend (React)
cd ..
npm install
npm start
# Active interface at http://localhost:3000
```
